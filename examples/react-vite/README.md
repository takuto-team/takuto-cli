# React + Vite Preset

Ready-to-use Maestro configuration for React projects using Vite.

## What's included

- `npm ci` as the install command
- **Run commands**: Dev Server, Storybook, Preview Build
- Claude Code skills from `morphet81/cheat-sheets` (installed before each workflow)
- Dynamic port forwarding for dev servers (Vite on 5173, Storybook on 6006, etc.)

## Setup

```bash
cp .maestro/config.toml /path/to/your/maestro/.maestro/config.toml
```

Edit the values marked with `←` (repo URL, branch, ticketing system).
